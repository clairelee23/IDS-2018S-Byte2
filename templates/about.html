Accountability: About This Data Set

This data set is the PM2.5 Data of Five Chinese Cities Data Set. This data set contains time series data for hourly PM2.5 data in Beijing, Shanghai, Guangzhou, Chengdu and Shenyang, which are five main cities in China.

Where did this data come from?

This data set comes from UCI Machine Learning Repository, which is a open collection of databases, domain theories, and data generators with a high reputation for high-quality data.

There are five separate original data sets, each for one main city in China. After being downloaded from the original source, these five data sets were cleaned up according to the following steps:

- 1. Since specifically, we want to explore the trend of PM2.5 at 8 a.m. in China, the irrelevant data like Cumulated Hours of Rain, Temperature, etc. were deleted at first. Only the potentially useful data, i.e., date data, season data and PM2.5 data of US Embassy in each data set were reserved.

- 2. Then the five data sets were combined into the integrated data set as shown on the Home page.

Who has access to it?

It is a public repository with transparency. Anyone can download it from here, but only people with premission can modify it.
