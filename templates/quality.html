Quality: About This Data Set

Completeness: Even there are some missing data in this data set, but they are less than 1%. Thus, this data set has a good completeness.

Coherence: The data make sense relative to itself. Outliers are explainable and reasonable. And the PM2.5 values have a meaningful distribution.

Correctness: This data set is reliable.

Is the data complete?

The trend of PM2.5 of the five main cities in China during 2013 to 2015 can be shown by the chosen data set. However, as shown in Figure 1, there are still some missing data.

But the missing data only take a tiny part of the whole data set, they would not cause a significant effect on the analysis. There might be several reasons for the missing data, like broken devices, incorrect recordings, etc., but the real causes are unclear.

Figure 1. Missing rate

Is the data coherent?

All the PM2.5 values in the data set are numbers, which are meaningful data.

Figure 2 shows the boxplots of the PM2.5 values. But there seems to be too many outliers, which goes beyond my expectations. Thus, when I checked these outliers by comparing these outliers with real reports from China Meteorological Administration, which is the authoritative organization in China, they are consistent. That made me realize that these outliers might be caused by the degradation of environment. And the fact that these outliers are all distributed above the average value verified it.

Histograms in Figure 3 show that PM2.5 values of each city have the similar distribution, which make sense.

Is the data correct?

Since the data set comes from UCI Machine Learning Repository, it should be solid and realiable.

Look into the data, however, the values of PM2.5 are all integers, so they might be reported in round numbers. And there is no way to check whether the measurement of the data is right or not. 30 random values were chosen and compared to China Meteorological Administration to verify their correctness, and they are consistent.

In my opinion, it is hard for us to verify the data from the very begining like how it is collected and cleaning in the real world, especially when the data set is large. Thus, what we can do is to find the most trustful data set.
